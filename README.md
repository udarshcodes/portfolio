# Udarsh Goyal's Personal Portfolio Website

This is my personal portfolio website.  
It showcases my education, experience, projects, skills, certifications, and official badges in a clean, modern single-page layout.

## 🚀 Live Preview

> https://udarshcodes.github.io/personal_portfolio/

## 🏗️ Tech Stack

- **HTML5** – Semantic single-page structure.
- **CSS3** – Custom responsive layout & styling (`style.css`).
- **JavaScript (vanilla)** – For interactivity (navigation, canvas background, sliders etc.) via `script.js`.

No frameworks (React/Vue/etc.) — just clean HTML/CSS/JS.
